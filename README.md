
# VNPAY Client Package

## Introduction

Show up VNPAY screen just use a single function.

## Installation

Add using pub add command:

```bash
    flutter pub add vnpay_client
```

## Usage/Examples

```dart
showVNPayScreen(
   paymentUrl: paymentUrl,
   onPaymentSuccess: (value) {}, 
   onPaymentError: (error) {},
);
```
